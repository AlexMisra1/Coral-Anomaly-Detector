Coral Anomaly Detection

This Coral Anomaly Detector that I created uses advanced machine learning algorithms to analyze images of corals/coral reefs. By creating this tool to detect anomalies in coral formations, I wanted to help researchers and conservationists monitor how bleached corals are in order to take action to protect vulnerable ecosystems.
